## üöÄ Getting Started

### Prerequisites

Before you begin, ensure you have:

- ‚úÖ Active HubSpot account with CMS Hub access
- ‚úÖ Access to HubSpot Design Manager
- ‚úÖ Git installed on your local machine
- ‚úÖ Basic understanding of HubSpot CMS concepts

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/hhm150456/hubspot-personal-portofolio-website.git
   cd hubspot-personal-portfolio-website
   ```

2. **Open HubSpot Design Manager**
   - Log in to your HubSpot account
   - Navigate to Marketing ‚Üí Files and Templates ‚Üí Design Tools

3. **Create a new project folder**
   - Click "File" ‚Üí "New Folder"
   - Name it "Personal Portfolio"

4. **Upload files**
   - Upload all repository files to your newly created folder
   - Maintain the folder structure

5. **Review module settings**
   - Check each module's fields and default values
   - Adjust settings as needed

## üìñ Usage

### Creating a New Page

1. Navigate to **Marketing ‚Üí Website ‚Üí Website Pages**
2. Click **Create ‚Üí Website Page**
3. Select the portfolio template
4. Use the drag-and-drop editor to add sections and modules
5. Populate content through the module editor
6. Preview for responsiveness
7. Publish when ready

### Editing Content

All content can be edited directly in the HubSpot page editor:

- **Text**: Click and type or use the rich text editor
- **Images**: Click to upload or select from file manager
- **Links**: Update URLs in module settings
- **Layouts**: Adjust using drag-and-drop interface

### Customizing Modules

Each module includes editable fields accessible through the module editor:

- Repeater fields for adding/removing items
- Toggle fields for showing/hiding elements
- Choice fields for layout options
- Color pickers for styling
- Image uploaders for media
